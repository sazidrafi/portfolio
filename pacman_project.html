— Overview

Project Description

A faithful recreation of the iconic Pac-Man arcade game using JavaScript and HTML5 Canvas. Guide Pac-Man through a maze to eat pellets while avoiding ghosts. Features original gameplay mechanics, autonomous ghost behavior, and score tracking.

This project demonstrates advanced game development skills including AI pathfinding, collision detection, sprite animation, and game state management - all built with vanilla JavaScript without external game engines.

— Gameplay

Core Gameplay Features

Maze Navigation

Navigate through a complex procedurally drawn maze to eat all pellets and complete the level.

Ghost AI

4 unique ghosts (Blinky, Pinky, Inky, Clyde) with different personalities and chase patterns.

Power Pellets

Eat power pellets to temporarily turn the tables and chase the frightened ghosts for bonus points.

Score System

Collect pellets and bonuses for points. Track your high score across gameplay sessions.

— Features

Key Technical Features

Maze Generation

Procedurally drawn maze with walls, pellets, and power pellets creating an authentic Pac-Man experience with proper collision detection and boundary checking.

Advanced Ghost AI

  • Blinky (Red): Aggressive chaser that directly pursues Pac-Man using shortest path algorithms
  • Pinky (Pink): Ambusher that tries to position ahead of Pac-Man's current direction
  • Inky (Cyan): Unpredictable with mixed behavior patterns combining chase and scatter modes
  • Clyde (Orange): Alternates between scatter and chase modes based on distance to Pac-Man

Pac-Man Mechanics

  • Smooth directional movement with responsive keyboard controls
  • Animated mouth opening and closing during movement for authentic feel
  • Precise collision detection with walls and ghost encounters
  • Wraparound tunnel navigation at screen edges

Scoring System

  • Regular Pellets: 10 points each for standard maze navigation
  • Power Pellets: 50 points each for temporary ghost vulnerability
  • Frightened Ghosts: Bonus points when caught during power-up mode
  • Fruit Bonuses: Special items for extra score opportunities

Win/Lose Conditions

  • Victory: Clear all pellets from the maze to advance to the next level with increased difficulty
  • Game Over: Lose all three lives when caught by ghosts, displaying final score

— Development

Technical Challenges & Solutions

Challenge: Ghost Pathfinding

Solution: Implemented a hybrid approach combining A* algorithm for intelligent chasing and random movement for scatter mode. This creates realistic ghost behavior patterns that feel authentic to the original game while being computationally efficient.

Challenge: Collision Detection

Solution: Used grid-based checks for maze walls with pixel-perfect hitboxes for ghost encounters. This dual-system approach ensures smooth gameplay while maintaining accurate collision detection for both static and dynamic objects.

Challenge: Sprite Animation

Solution: Created frame-by-frame animation cycles for Pac-Man and ghosts using requestAnimationFrame. This provides smooth, consistent animations at 60 FPS with proper timing and sprite sheet management.

Challenge: Game State Management

Solution: Implemented finite state machines (FSM) for managing ghost behaviors (chase, scatter, frightened) and game states (playing, paused, game over). This modular approach makes the codebase maintainable and extensible.
